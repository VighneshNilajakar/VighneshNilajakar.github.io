<h3 id="introduction">Introduction</h3>
<p align= "left" style="font-size:20px; padding: 10px;"><a href="https://github.com/VighneshNilajakar/Grocery_Shop_Management_System" target="_blank" style="color : white">Grocery Shop Management System</a> is a Software made to ease the management of a Grocery Shop and assists in keeping logs of Products, Customers, Inventory, and Bills. This Software can run on most CPUs, Its deployment is easy, and the UI is CLI based.</p>
<h3 id="programs_used">Programs used</h3>
<p align= "left" style="font-size:20px; padding: 10px;">It uses Python for Menu, Data Input, and to print the Output of the Data, and MySQL is used to store the Data entered. MySQL Connector/Python is also used to Integrate Python and MySQL. Basic knowledge of Python was used while Coding the Project.</p>
<h3 id="deployment">Deployement</h3>
<p align= "left" style="font-size:20px; padding: 10px;">The Deployment process is very Easy, Considering you have installed <a href="https://wiki.python.org/moin/BeginnersGuide/Download" target="_blank" style="color : white">Python</a>, <a href="https://dev.mysql.com/downloads/" target="_blank" style="color : white">MySQL</a> and <a href="https://dev.mysql.com/doc/connector-python/en/connector-python-installation-binary.html" target="_blank" style="color : white">MySQL Connector/Python</a> ( If not click on each Software to See Installation Guides ). Git Clone the Repository via Terminal, with this Command.</p>
<code style="padding: 10px; font-size: 90%">
git clone https://github.com/VighneshNilajakar/Grocery_Shop_Management_System.git
</code>
<p align= "left" style="font-size:20px; padding: 10px;">After Cloning, Open <code>INITIALIZATION.py</code> and run it. either you can run it with a Text Editor or from The Terminal.<br>
CAUTION : This Script clears all Data from the Database, If it already Exists.</p>
<code style="padding: 10px; font-size: 90%">
python3 /path/to/file/Grocery_Shop_Management_System/INITIALIZATION.py
</code>
<p align= "left" style="font-size:20px; padding: 10px;">Enter 'Y' or 'y if you want to run the Script, Add your MySQL Username and Passwords, and It will create all necessary Databases.<br>
Similarly, Run <code>MAIN.py</code> with a Text Editor or in Terminal with this code.</p>
<code style="padding: 10px; font-size: 90%">
python3 /path/to/file/Grocery_Shop_Management_System/MAIN.py
</code>
<p align= "left" style="font-size:20px; padding: 10px;">After you Sign Into MySQL with your Credential, The CLI-based Menu is pretty clear and you can use it to maintain Product Data, Customers Data and Manufacturers Data.</p>
